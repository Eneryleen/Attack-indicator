AttackIndicator is a modern, lightweight damage indicator plugin for Paper servers. Display beautiful animated floating damage numbers when entities take damage, powered by Display Entities for smooth performance and visual appeal.

**Key Features:**
- **Smooth Animations** - Damage indicators float upward with configurable speed and duration
- **Highly Customizable** - Full control over appearance, size, position, and behavior
- **MiniMessage Support** - Rich text formatting with gradients, colors, and styles
- **Smart Filtering** - Choose when to show indicators: all damage, player-only, or exclude self-damage
- **Player Damage Support** - Optional indicators on players (configurable)
- **Entity Filtering** - Whitelist/blacklist specific entity types
- **Random Positioning** - Natural look with slight random offsets to prevent stacking
- **Auto-Update Checker** - Get notified when new versions are available
- **World Management** - Disable indicators in specific worlds
- **Performance Optimized** - Uses Display Entities, minimal server impact

### Commands
- `/ai reload` - Reload plugin configuration
